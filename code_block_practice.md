## How to use code block in Markdown

we usually use like this :

```
print("Hello")
```

But we can emphasize programming language like this :

```python
print("hello")
```

If you want to emphasize more language

* Bash -> bash
* C# -> cs
* C++ -> cpp
* Diff -> diff
* HTML, XML -> html
* HTTP -> http
* Ini -> ini
* Json -> json
* Java -> java
* JavaScript -> javascript
* PHP -> php
* Perl -> perl
* Python -> python
* Ruby -> ruby
* SQL -> sql

More example :

```bash
git init
```

```
System.out.println("Hello World");
```
